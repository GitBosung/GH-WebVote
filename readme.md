# ghonlinevoteweb
학급 반장선거를 웹으로 대신 한다.


2018/9/01 학급자치회장 선거 전자투표로 진행

문제점 발생 -- 

1. 중복투표 가능 
    쿠키 초기화, IP우회를 하면 n번 가능.
    
2. 투표데이터 정리 불가



2019/3/10 문제점 해결후 한번더 전자투표 진행

문제점 해결 -- 

1. 중복투표 가능

    유니코드를 생성, 투표 직전 무작위 배부 
    
    유니코드는 1회용, 투표시 데이터베이스에 저장. 
    
2. 득표수 시각화 


    막대그래프사용, 보기 편하도록 시각화.
